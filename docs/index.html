Welcome to Data Girls Homepage

This project explores the factors that influence job transitions and salary trends in the data science field. Through interactive visualizations, we analyze how education, experience, company characteristics, and demographics impact both career mobility and compensation.

Using data from 2020 to 2023, we highlight patterns across roles such as Data Scientist, Machine Learning Engineer... revealing how these evolve over time, differ by country, and vary with experience level.
